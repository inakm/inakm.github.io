## 📊 Business Intelligence & Analytics Dashboards
Built completely utilizing client-side Javascript logic, leveraging `PapaParse` and `Chart.js` to ingest flat `.csv` multidimensional data natively in the web browser. The visual design is powered by **Tailwind CSS**.

- **[Prism Dashboard](https://inakm.github.io/page/projects/dashboards/Prism.html)**: Interactive regional sales and segment trend BI view.
- **[RMM Analytics](https://inakm.github.io/page/projects/dashboards/RMM.html)**: Advanced analytics dashboard showcasing custom dataset table joins, Quarter/Month time-series financial breakdowns, state-level geography mapping, and top-tier customer reporting.
- **[VEVOi Insights](https://inakm.github.io/page/projects/dashboards/VEVOi.html)**: Geographic unit-sells financial breakdown interface.
- **[PowerBI Models](https://inakm.github.io/page/projects/powerbi.html)**: Features full downloadable `.pbix` relational BI models.
