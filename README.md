- **PowerBI Hub**: Features comprehensive downloadable `.pbix` relational models including *FM Power*, *Prism Real Estate*, and global *Billionaires Statistics*.

## 💼 Core Finance & AI Engineering
- **Advanced Excel Modeling Hub**: Corporate accounting templates including DCF scaling, Fixed Asset Registers, AR Aging Reports, and an *AI Fin-Model Audit* showcase.
- **AI Recruitment Agents**: Advanced structured prompt generators allowing non-technical recruiters/users to spin up highly verified CV targeting and job scraping bots.
- **ACCA (F9/F7) Architecture**: Trackers and modeling frameworks directly addressing Section-C constructions for strategic financial management.
- **Corporate M&A / Equity Research**: Extensive equity breakdowns, pitch infrastructure (GatiGo), and BQL (Bloomberg Query Language) macro forecasting routines.
