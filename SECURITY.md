# Security Policy

## Supported Versions

Please see the following list of active and supported versions of this project.

| Version | Supported          |
| ------- | ------------------ |
| v1.0.x  | :white_check_mark: |
| < v1.0  | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within this project, please send an e-mail to Anjani Kumar Mishra via [inakm@proton.me](mailto:inakm@proton.me). All security vulnerabilities will be promptly addressed.

Please do **not** report security vulnerabilities through public GitHub issues.

### Disclosure Policy

- We will acknowledge receipt of your vulnerability report the next business day and strive to send you regular updates about our progress.
- If you're curious about the status of your report, please feel free to email again.
- Once the vulnerability is resolved, if applicable, we will notify you before the patch release so you can test the solution.
- We request that you not publicly disclose a vulnerability until we have had the opportunity to address it natively.

Thank you for reporting any issues directly.
